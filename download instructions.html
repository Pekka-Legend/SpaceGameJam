Download Instructions


1) Click on a game that has a (Download) tag next to the name

2) Click on the zip file in the bottom right corner of your screen to open it

3) Click extract all at the top of the opened file, then click extract

4) Download python if you don't have it already

5) While in the folder that contains the game, click on the bar that says Download > Game (Game should be whatever game you downloaded)

6) Type "cmd", hit enter

7) When the command prompt opens, type "python filename.py"
